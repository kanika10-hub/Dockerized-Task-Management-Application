# Dockerized Task Management Application

##  Overview
A containerized task management application built using modern DevOps practices. The project leverages Docker to ensure consistent environments, simplified deployment, and efficient application management across different systems.

---

##  Features
- Task creation, update, and deletion (CRUD operations)
- Containerized application using Docker
- Consistent environment across development and deployment
- Simplified setup and execution using Docker containers
- Improved scalability and portability

---

##  Tech Stack
- Docker
- Node.js
- JavaScript
- HTML
- CSS

---

##  DevOps Highlights
- Created Docker images for the application
- Ensured **100% environment consistency**
- Eliminated dependency conflicts across systems
- Reduced setup time significantly using containerization
- Practiced environment isolation and deployment workflows
